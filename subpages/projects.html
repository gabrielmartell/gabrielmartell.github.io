Here are the following projects from my academic studies, TAShip and personal.

The COMP3005 Autograder was a tool for student submissions of Project v1 of the Carleton University COMP3005 Winter 2024 Class. It was an automated system that would parse through each student's github repository submission, read their database dump file to fabricate a new database, and records the query metrics to determine correctness and efficiency - created on a Linux environment. The autograder itself is currently private to withold Academic Integrity, but the submission script template and further information for students can be found on the image's hyperlink.
Hunter of Monsters is a RTS game made for the Carleton University COMP4501 Final Project. It was created on GODOT 4.0, where you must harvest resources and supplies and build your bases, create soldiers and fabricate traps to survive a enemy-wave system that gets progressively harder.
Hungry Man is a Horror game developed for the Carleton University COMP3501 Final Project. You are tasked to explore around the map finding collectibles to score points while avoiding "Hungry Man" by hiding in bushes or safe spots. This was developed using C++.
The Way of Blood is a top-down driving/shooter game developed for the Carleton University COMP2501 Final Project. You are tasked to drive all the way to Canada while avoiding enemies along the way. You are tasked to use power-ups and new weaponry to accomplish this. This was developed using C and compiled with CMake.
This Elevator Simulator was developed for Carleton University COMP3004 Final Assignment. This simulator takes a perspective of the control center with given parameters, such as number of floors, number of elevators, etc. This was developed on QtCreator in a Linux environment. You can find my presentation video here.
Heartwave is a AED Device Simulation for Carleton University COMP3004 Final Project. This simulation is a dynamic monitor of your heart rate with different aspects such as heart rate charting and battery life consumption. This was developed in QtCreator in a Linux environment.
Look-Inna-Book is a virtual library bookstore simulation, developed for Carleton University COMP3005 Final Project. This simulation allows you to take control of a customer or an administrator of the system - heavily utilizing SQL databases.
